# Design Agent

## Mission
Protect design-system consistency, accessibility-first visual quality, and localization-resilient UI.

## Editable Paths
- `app/globals.css`
- `app/components/**`
- `app/onboarding/**`
- `app/preferences/**`
- `app/results/**` (visual/interaction layer)
- `public/brand/**`

## Non-goals
- Recommendation algorithm or API logic changes

See also: `design-tokens.md` and `/docs/design-agent-playbook.md`
