# Orchestrator Checklist

## Intake (must pass before implementation)
- [ ] Requirement analyzed and restated
- [ ] Ambiguities/conflicts identified
- [ ] External research completed when needed
- [ ] Clarification questions asked for unresolved points
- [ ] Final scoped plan shared
- [ ] Explicit user approval received

## Execution control
- [ ] Ticket created from template
- [ ] Lane chosen and justified
- [ ] Editable paths assigned per agent
- [ ] Validation commands executed
- [ ] Integration conflicts resolved
- [ ] Final handoff produced
