# Growth Checklist

Use before global rollout changes.

## i18n
- Language routing and copy quality checked for `ko`, `en`.
- Locale formatting checked for distance/number/date.

## SEO/GSC
- Canonical + hreflang verified.
- Sitemap + robots verified.
- Search Console property and sitemap submission tracked.

## Ads
- Landing page destination health checked.
- Policy-sensitive copy/content reviewed.
- Consent handling verified for required regions.
