# Meal Recommendation Web (TypeScript)

Location-based lunch/dinner recommendation MVP.

## Stack
- Next.js (App Router)
- TypeScript
- API Route Handlers

## Run
```bash
npm install
npm run dev
```

Open: http://127.0.0.1:3000

## API
- GET `/api/availability`
- POST `/api/recommendations`
- POST `/api/events`

## Env
Copy `.env.local.example` to `.env.local`.
